# 💍 Shaadi Ki Yadgaar Gallery

Ek khoobsurat wedding photo gallery website — bilkul free, koi server nahi chahiye!

## 📁 File Structure

```
wedding-gallery/
├── index.html          ← Main page (yahan se open karein)
├── css/
│   └── style.css       ← Poora design & styling
├── js/
│   └── gallery.js      ← Sari functionality
└── README.md           ← Yeh file
```

## 🚀 Kaise Use Karein

1. **ZIP extract karein** apne computer mein
2. **index.html** double-click karein — browser mein khul jayega
3. **Photos upload karein** aur categories choose karein
4. **Ho gaya!** 🎉

## ✨ Features

- **Masonry grid layout** — photos khubsoorat tarike se dizplay hongi
- **Category filter tabs** — kisi bhi function ki photos akele dekh saktey hain
- **Lightbox viewer** — photo par click karke badi kar ke dekhein
  - ← → arrows se navigate karein
  - Keyboard arrows bhi kaam karte hain
  - Touch swipe mobile par bhi kaam karta hai
- **Drag & drop upload** — photos seedha drop karein
- **Multiple upload** — ek saath kai photos upload karein
- **Delete option** — galat photo hata sakein
- **Stats bar** — total aur filtered photos count

## ⚠️ Note

Photos browser session mein rehti hain — page refresh karne par photos chali jayengi.
Agar permanently save karna ho toh kisi web host (Netlify, GitHub Pages) par deploy karein.
